Términos y condiciones del servicio de planes mensuales


Activación del servicio

El servicio se activa una vez realizado el pago mensual anticipado. A partir de ese momento, comienza a contar el periodo de 30 días del plan contratado.


Modalidad de trabajo

El servicio funciona bajo un sistema de solicitudes activas.
Cada plan incluye un número específico de solicitudes que pueden trabajarse de forma simultánea. Las solicitudes se atienden en orden y se entregan de manera continua durante el mes.


Entregas y tiempos

Los tiempos de entrega estimados son:
Piezas gráficas: 24–48 horas hábiles Reels: 2–4 días hábiles
Estos tiempos pueden variar según la complejidad de la solicitud o el flujo de trabajo.


Solicitudes y contenido

El cliente debe proporcionar:

  • Textos (si aplica)
  • Material audiovisual (fotos, videos)
  • Referencias o indicaciones claras

La calidad del resultado final depende directamente del material suministrado.


Cambios y revisiones

Cada pieza incluye hasta 2 rondas de ajustes.
Cambios adicionales o solicitudes fuera del alcance inicial pueden requerir tiempo adicional o un costo extra.


Alcance del servicio

El servicio incluye:

  • Diseño de piezas gráficas
  • Edición de reels cortos (maximo de 45 segundos)

No incluye:

  • Gestión o administración de redes sociales
  • Publicación de contenido
  • Pauta o publicidad

Uso del plan

El plan es válido durante 30 días desde su activación.
Las piezas o solicitudes no utilizadas dentro de ese periodo no son acumulables para el siguiente mes.


Pagos y renovación

El servicio se maneja bajo pago mensual anticipado.
La renovación del plan debe realizarse al finalizar el periodo activo para continuar con el servicio sin interrupciones.


Pausas o cancelación

El cliente puede decidir no renovar el plan para el siguiente periodo.
No se realizan reembolsos una vez iniciado el ciclo mensual.


Comunicación

La comunicación se realiza a través de los canales definidos (WhatsApp o correo).
Para mantener un flujo ordenado, las solicitudes deben enviarse mediante el formato o sistema establecido.
